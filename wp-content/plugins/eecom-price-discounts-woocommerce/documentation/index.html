Price and discounts based on Custom Roles in WooCommerce

The plugin allows you to create custom roles in woocommerce. This is not a default feature. The plugin also allows you to set discounts of a product based on custom roles and then give discount based on roles. Will help store owners increase sales based on customer groups.

    • Introduction

    The plugin allows you to create custom roles in woocommerce. This is not a default feature. The plugin also allows you to set discounts of a product based on custom roles and then give discount based on roles. Will help store owners increase sales based on customer groups.

     

    Features

    • Allows to create numbers of Users roles
    • Delete particular user role at any time
    • Set different products prices for all roles

    Feature Improved

    • Discount price for variable Products :: Feature allows admin to add a discount price for each user role under all variations in variable products

    New Feature added

    • Percentage discounts for custom users roles.

    # version Release: 3.0

    Compatibility Test :

    • Plugin Compatibility tested with Wordpress 4.3.1
    • Plugin Compatibility tested with WooCommerce 2.4.10
    • Feature are completely working with Latest Version of Wordpress and WooCommerce

    Setup

    Go to Plugins > Add New > Upload Plugin.

    Install plugin by choosing the plugin zip file and click on Upload link.

    If successful, click Activate Plugin to activate it in the ‘Plugins’ menu in WordPress

    Ensure 'Integration' option under WooCommerce settings.
