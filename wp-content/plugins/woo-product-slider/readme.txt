== Installation ==

Installing this plugin as regular WordPress plugin.

After install, you are ready to use Product Slider in your post, page, widget or anywhere you like. How to create Product Slider? lets see.


1. Upload the folder woo-product-slider to "/wp-content/plugins/" '
2. Activate the plugin through the "Plugins" menu in WordPress .
3. Call shortcode to your WordPress editor directly, by using 
`
[woo-product-slider id="01" title="Latest Product"]
`
Call shortcode to your php template file using 
`
<?php  echo do_shortcode('[woo-product-slider id="01" title="Latest Product"]'); ?>
`
